We are emailed a file every week that contains terminated users and their assets (laptops, phones, etc). We add these users to a tracker file. Then we take that file and we look up the associated UPS label PDF. We take the UPS ID and search in the UPS website for the status. We update the status in the tracker. If the asset is not being returned we send the manager an email reminder. The following week if still unreturned we send the user an email. And finally the last week we send an email to the ECC mailbox.
